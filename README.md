# Zukunftsentscheid Website

A Next.js project with MDX support for [Hamburger Zukunftsentscheid](https://zukunftsentscheid-hamburg.de/).

## Quick Start

```
bun install
bun dev
```

Visit [http://localhost:3000](http://localhost:3000)
